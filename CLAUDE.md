# frevia-lp（配信者募集ランディングページ）

## これは何か
配信事務所Freviaの「ライバー（配信者）募集」ページです。静的な1枚のHTMLページで、
GitHub Pages（GitHubの無料ホスティング機能）で公開しています。

公開URL: `https://recruit.frevia-agency.jp`

## 公開方法
`master` ブランチへ変更を `git push` すると、GitHub Actions（自動処理の仕組み）が
自動的に本番サイトへ反映します。手作業での「デプロイ」操作は不要です。

```
git add -A
git commit -m "変更内容"
git push
```

反映まで数十秒〜数分かかることがあります。

## 応募フォームのバックエンド（受付処理）について
このページの応募フォームが送信したデータを受け取り、データベースへの保存と
管理者への通知メール送信を行っているのは、別プロジェクトです。

- 場所: `~/Claude/apps/frevia-recruit-api`
- コードを変更する場合、このリポジトリ（frevia-lp）はフォームの見た目と
  送信処理（送信ボタンを押した時の動き）のみを持ち、実際の受付処理は上記の
  別プロジェクト側にあります。

## 応募データの確認方法
応募内容（お名前・メールアドレスなど）の確認コマンドは
`~/Claude/apps/frevia-recruit-api/CLAUDE.md` を参照してください。
